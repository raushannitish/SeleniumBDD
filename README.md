# DemoBlaze End-to-End Automation with Behave and Selenium

This project contains automated test scripts for the [DemoBlaze](https://www.demoblaze.com/index.html) web application using:
- Python
- Selenium WebDriver
- Behave (BDD)
- HTML Reports
